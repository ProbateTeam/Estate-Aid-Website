# Estate Aid Network Theme

Custom standalone WordPress theme scaffold for The Estate Aid Network.

## Included

- Custom responsive header with sticky navigation and dropdown menus
- Custom homepage template in `front-page.php`
- Custom footer with four columns, newsletter-ready area, and legal links
- Theme Customizer fields for social URLs
- Footer widget areas and flexible homepage widget areas
- Theme assets in `assets/css` and `assets/js`

## Install Into WordPress

1. Copy the `estate-aid-network` folder into `wp-content/themes/`
2. Activate **Estate Aid Network** in WordPress Appearance > Themes
3. Assign menus in Appearance > Menus
4. Set a static homepage in Settings > Reading if needed
5. Configure social links in Appearance > Customize
